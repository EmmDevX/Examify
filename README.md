# Examify – Online Quiz & Exam Preparation Platform

Examify is a full-stack web application that helps students prepare for exams through interactive quizzes, real-time scoring, performance tracking, and an admin dashboard for managing quizzes and questions.

## 🚀 Features

### Student Features
- User registration and login system
- Take quizzes by subject
- Timed quizzes with auto-submit
- Instant scoring and results
- Detailed answer review
- Personal performance tracking dashboard
- Leaderboard system

### Admin Features
- Admin dashboard
- Create, edit, and delete quizzes
- Add and manage questions
- View users and quiz statistics
- Monitor attempts and performance

- Users
- Quizzes
- Questions
- Subjects
- Attempts
- Answers
